# Playbook Operacional de Vendas

## Rotina diária
1. Prospectar novos leads (lista local + indicação).
2. Realizar demos agendadas.
3. Follow-up dos trials ativos.
4. Registrar status no CRM.

## Cadência de follow-up
- D+1: mensagem de confirmação de valor.
- D+3: revisão de uso e objeções.
- D+7: proposta comercial.
- D+12: decisão final antes do fim do trial.

## Objeções comuns
- "Já uso planilha": mostrar tempo gasto e perda de controle.
- "Está caro": comparar com custo de retrabalho/no-show.
- "Minha equipe não usa": oferecer onboarding guiado.

## Definição de done para cliente ativo
- 1 usuário admin ativo.
- 10+ clientes cadastrados.
- 5+ agendamentos registrados.
- 1 venda registrada.
