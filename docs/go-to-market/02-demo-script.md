# Script de Demo (20 minutos)

## 1) Abertura (2 min)
- Entender cenário atual da oficina.
- Confirmar dor principal: agenda, retorno de clientes ou visão financeira.

## 2) Fluxo principal (12 min)
- Login e visão geral.
- Cadastro rápido de cliente e veículo.
- Criação de agendamento.
- Registro de venda e despesa.
- Dashboard com leitura de resultado.

## 3) Valor de negócio (4 min)
- Redução de no-show com confirmação de agenda.
- Mais retorno de clientes pela base organizada.
- Decisão melhor com números de vendas vs despesas.

## 4) Fechamento (2 min)
- Convite para trial de 14 dias.
- Definir data de check-in (D+3).
- Definir meta de sucesso do trial.
